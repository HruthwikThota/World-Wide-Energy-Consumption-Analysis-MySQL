# 🌍 World Wide Energy Consumption Analysis using MySQL

## 📌 Project Overview

This project analyzes worldwide energy production, consumption, GDP, population, and CO₂ emissions using MySQL.

The project demonstrates SQL concepts such as Joins, Aggregate Functions, Subqueries, GROUP BY, ORDER BY, Foreign Keys, and Data Analysis Queries to generate meaningful business insights from global energy datasets.

---

## 🎯 Problem Statement

Understanding global energy consumption and emissions is essential for sustainable development. This project analyzes worldwide energy data to identify trends, compare countries, and derive useful insights using SQL.

## 📈 Project Analysis

The project answers 18 analytical business questions including:

- Total CO₂ emissions by country
- Top GDP countries
- Energy production vs consumption
- Global emission trends
- Population vs emissions
- GDP vs energy production
- Emission-to-GDP ratio
- Consumption per capita
- Production per capita
- Global emission share
- Top populated countries
- Global yearly comparisons
